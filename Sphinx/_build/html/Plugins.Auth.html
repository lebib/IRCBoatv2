    class Plugins.Auth.User.User(line)¶

    Bases: object

    User are cosidered as entries inside a csv file manipulated by UserList.

    Parameters:line (str) – the line who’ll be parsed.

    The content of line is based on : LOGIN,PASSWORD,LEVEL,NICKNAME,HOST,

    The hashing, salting and encrypting is made by sha256_crypt from the passlib library.

    Example:"myLogin,myPassword,0,myCurrentNickname,myCurrentHost"

    class Plugins.Auth.User.UserList(source)¶

    Bases: dict

    UserList is based on dict in order to have the username as the key and the user object as a value.

    The main goal here is to have __init__ loading source and gather all the users from it.

    Parameters:source (str) – CSV file where are stored the users entries.
