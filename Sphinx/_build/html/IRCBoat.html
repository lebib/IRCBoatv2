    IRCBoat module¶

    class IRCBoat.Connection(host, port, events, encoding, ssl, verbose)¶

    Bases: bottom.connection.Connection

    Connection is only a specialization of bottom.connection.Connection. The idea behind it it’s to have access directly on the prompt all the messages from the server in order to easily debug the application. In the future it’ll be print in a log file.

    Parameters:
    • host (str) – Host of the IRC server
    • port (int) – Port of the IRC server
    • encoding (str) – Encoding used to get the IRC messages
    • ssl (bool) – If using SSL or not
    • verbose (bool) – If printing all the server message or not
